# Executive-Self-Service-BI-AI
### AI-Powered Portfolio Analytics Platform (Streamlit + DuckDB + LLM SQL Engine)

---

## Overview

Executive AI Self-Service BI is an experimental analytics platform designed to simulate a **next-generation Asset Management Intelligence System**.

The project combines:

- Executive-level dashboards
- AI-generated SQL queries
- Self-service analytics
- Real-time visualisation

Unlike traditional BI tools, this application allows users to **ask business questions in natural language** and instantly receive:

- SQL queries generated by AI
- Data results
- Auto-generated visualisations

The goal is to demonstrate how modern ManCo / Asset Management platforms could evolve beyond static dashboards.

---

## Key Value Proposition

This project explores a future where:

- Executives no longer depend on analysts to build reports
- Data exploration becomes conversational
- Dashboards adapt dynamically to business questions

## Analytics Engine (DuckDB)

DuckDB is used as the analytical layer because it provides:

- Zero-config local execution
- Perfect integration with Python & Streamlit

A unified analytics view is created:

allowing fast executive queries.

## AI Self-Service Layer

Users can ask business questions such as:

"Monthly fee trend by strategy"
"Which department has the highest HR cost?"
"Top funds by net flow this year"


The app will:

1. Generate SQL via AI
2. Execute against DuckDB
3. Render an executive-style visual automatically

---

## How It Works (Technical Flow)

1. User writes a question
2. `generate_sql()` sends context + schema to LLM
3. SQL is returned and cleaned
4. DuckDB executes query
5. App auto-detects chart type

No manual dashboard design required.
